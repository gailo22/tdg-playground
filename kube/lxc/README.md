## LXC/LXD on Manjaro

### installation
```
$ sudo snap install lxd
$ sudo lxd init
$ sudo lxc launch ubuntu:20.04 ubuntu-2004
$ sudo lxc list
+-------------+---------+----------------------+-----------------------------------------------+-----------+-----------+
|    NAME     |  STATE  |         IPV4         |                     IPV6                      |   TYPE    | SNAPSHOTS |
+-------------+---------+----------------------+-----------------------------------------------+-----------+-----------+
| ubuntu-2004 | RUNNING | 10.46.100.103 (eth0) | fd42:2c93:7439:e07f:216:3eff:feae:5bdd (eth0) | CONTAINER | 0         |
+-------------+---------+----------------------+-----------------------------------------------+-----------+-----------+
$ sudo lxc exec ubuntu-2004 bash
root@ubuntu-2004:~#

```
